# django-onepage-website

This project is a personal portfolio website developed using Django.

## Features
- Personal biography
- Education
- Skills
- Languages
- Projects
- Contact information

## Technologies
- Python
- Django
- HTML
- CSS
- JavaScript

## Note
The frontend design is based on a publicly available HTML/CSS template. I integrated the template into Django, organized the template and static file structure, and customized the content for my personal portfolio.
